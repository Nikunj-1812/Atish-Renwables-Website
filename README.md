# Atish Website

Short description

This repository contains the frontend, admin panel, and backend for the Atish website — a marketing site with lead management, project listings, and an admin dashboard.

Quick start

- Install dependencies in each folder: `frontend`, `admin`, and `backend`.
- See each package.json for scripts to run the dev servers.
